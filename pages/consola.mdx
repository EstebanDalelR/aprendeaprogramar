# Consola

La consola también es llamada terminal o línea de comandos. Esta se usa para ejecutar comandos en el sistema.

## Abrir la consola

En esta ocasión, habrá diferencias entre tener un sistema Windows y uno IX (Mac o Linux), por ello debo diferenciarlos.

### Windows

#### CMD

Usa <kbd>Windows</kbd> y <kbd>R</kbd> para traer el programa <samp>Ejecutar</samp> y escribe `cmd`.  
![Instrucciones para abrir la consola, tecla windows y R, y escribir cmd](/consola/windows-r-cmd.png)

#### Powershell

Recientemente, Microsoft creó otra consola que se acerca más a la experiencia Linux, llamada Powershell. Recomiendo esta.
La puedes encontrar en el Menú escribiendo <samp>Powershell</samp>

![Instrucciones para abrir Powershell, menú y escribir Powershell](/consola/powershell.png)

#### WSL

Esta es mi absoluta favorita. Windows ahora ofrece la experiencia linux completa en su sub-sistema. Para acceder a WSL, [deberás seguir las instrucciones](https://docs.microsoft.com/es-es/windows/wsl/install). Curiosamente, empiezas desde Powershell, pero ya sabes llegar ahí.

### Mac

Mac tiene por defecto su terminal, que encuentras en Spotlight.  
Usa <kbd>Command</kbd> y <kbd>Espacio</kbd> y escribe <samp>terminal</samp>.  
![Instrucciones para abrir la terminal, command, espacio y escribir terminal](/consola/terminal.png)

### Linux

Si tienes Linux, ya sabes esto, pero dependerá de tu distribución.
