# Consola

La consola también es llamada terminal o línea de comandos. Esta se usa para ejecutar comandos en el sistema.
Desde acá los programadores controlan casi todo.

Lo más importante es no temerle.

## Abrir la consola

En esta ocasión, habrá diferencias entre tener un sistema Windows y uno IX (Mac o Linux), por ello debo diferenciarlos.

### Windows

#### CMD

Usa <kbd>Windows</kbd> y <kbd>R</kbd> para traer el programa <samp>Ejecutar</samp> y escribe `cmd`.

> Con WSL los comandos a usar serán los de Linux/Mac, tendrás un sistema IX, pero recuerda que los directorios pueden variar y no necesariamente se comunica directamente a tu pantalla

## Saber dónde estoy

La consola se navega por texto y es importante entender cómo funciona esto. Para el sistema operativo, todo es un archivo, incluso las carpetas. Las abstracciones que se crean vienen después, por eso es tan útil poder ver dónde estamos.

> Usaré carpeta y directorio de manera indiscriminada, evitaré _folder_ pero así también le dicen

### Windows

```
cd
```

En Windows usamos `$cd` sin parámetros para ver la posición actual. En inglés _change directory_, cambiar directorio tiene una pequeña complicación: solo aplica para el disco actual; si estás en `C:`, no podrás usarlo para ir a `D:`.

### IX

```
pwd
```

Para los IX usamos `pwd`. Este comando sólo ejecuta una función. En inglés _print working directory_, imprimir directorio de trabajo.

## Qué hay a mi alrededor

Una de las preguntas más importantes que pueden surgir es cuál es el contenido del directorio donde me encuentro. Este comando te responde y dependiendo de los parámetros que le pases podrías saber el tamaño, verlo como tabla, ver si es ejecutable, etc.

### Windows

```
dir
```

Este bocablo te responde con una lista de archivos y carpetas. En inglés _directory_, directorio.

### IX

```
ls
```

Su función es la misma que la anterior, pero en IX es `ls`. Su hermano es `la`, que te da más información.

## Moverme

Para ir a otro directorio y poder ver su contenido y ejecutar comandos desde otra locación. En este es importante el sentido de la barra, en Windows, anteriormente se usaba la barra invertida (`\`), ahora no.

### Windows

```
cd ruta\tan\larga\como\quieras

```

### IX

```
cd ruta/tan/larga/como/quieras

```

## Crear un archivo

### Windows

```
cd ruta\mi-archivo.txt
```

### IX

```
touch ruta/mi-archivo.txt
```

## Crear una carpeta

### Windows

```
md ruta\mi-archivo.txt
```

### IX

```
mkdir ruta/mi-archivo.txt
```

## Mover un archivo

### Windows

```
mv ruta\mi-archivo.txt ruta\mi-archivo-nuevo.txt
```

### IX

```
mv ruta/mi-archivo.txt ruta/mi-archivo-nuevo.txt
```

## Mover una carpeta

### Windows

```
mv ruta\mi-folder\ ruta\nueva\locacion\
```

### IX

```
mv ruta/mi-folder/ ruta/nueva/locacion/
```

## Eliminar un archivo

### Windows

```
del ruta\mi-archivo.txt
```

### IX

```
rm ruta/mi-archivo.txt
```

## Eliminar una carpeta

### Windows

```
rm ruta\mi-carpeta\
```

### IX

Usamos `-rf` para eliminar una carpeta, pidiendo que elimine su contenido con "recursividad" y "fuerza bruta".

```
rm -rf ruta/mi-carpeta/
```

## Cerrar la consola

### Windows

```
exit
```

### IX

```
exit
```
